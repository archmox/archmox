About Archmox

Archmox is an open-source virtualization stack for Arch Linux, ported from the Proxmox ecosystem by the AcreetionOS Project. It leverages Hyperbola GNU/Linux-libre as a stable, LTS-focused base with Arch Linux's LTS kernel carrying the necessary patches.

Archmox is a community-built port maintained by Natalie Cole-Clift Spiva and contributors. It is not affiliated with Proxmox Server Solutions GmbH.

What's Inside

🖧

Virtual Environment (PVE)

Full KVM virtualization and LXC containers with web management UI, live migration, snapshots, and template management.

pve-manager
💾

Backup Server (PBS)

Content-addressable, deduplicated backup system for VMs and containers. Written in Rust for performance and reliability.

proxmox-backup
✉️

Mail Gateway (PMG)

SMTP proxy with spam filtering, virus scanning, and policy routing to protect your email infrastructure.

pmg-api
🗄️

Software-Defined Storage

ZFS, Ceph, and LVM — pool local and networked storage into a unified, highly-available fabric with built-in replication.

ceph • zfs
🔗

HA Clustering

Corosync-based cluster stack with distributed resource management and automatic failover for production workloads.

corosync-pve
📦

Arch Linux Native

Every component packaged as PKGBUILDs. Install and update with pacman, manage with systemd. No .deb dependencies.

pacman -S archmox

Download

Archmox ISOs and binary repositories are available via cdn.archmox.acreetionos.org. Check the releases page for the latest builds.

An OpenBSD/HyperbolaBSD variant is also available at archmox-bsd.org.

Contribute

Archmox is community-driven. We accept pull requests on GitHub. Priority areas:

  • Porting Perl modules from Debian paths to Arch paths
  • Converting debian/rules to PKGBUILDs
  • Adapting pve-installer for archiso
  • Testing on bare metal
