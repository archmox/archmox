Binary Repository

Archmox packages are available via an Arch Linux binary repository. Add the following to your /etc/pacman.conf:

[archmox]
Server = https://cdn.archmox.acreetionos.org/repos/$arch

Then install with sudo pacman -Syu && sudo pacman -S archmox-pve.
